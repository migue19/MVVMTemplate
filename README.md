# Xcode Templates - Clean Architecture MVVM

A collection of Xcode templates implementing **Clean Architecture** principles with **MVVM** pattern and **Combine** for reactive programming.

## Updated Structure

The template has been restructured to align with the Clean Architecture implementation found in the `app-mobile-banking` project.

### Template Layers

Each scene generated by this template includes:

1. **Domain Layer**
   - `Entities/` - Business logic models
   - `Repositories/` - Repository protocols (contracts)
   - `UseCases/` - Business logic implementation

2. **Data Layer**
   - `Repositories/` - Repository implementation with data sources

3. **Presentation Layer**
   - `ViewModels/` - MVVM ViewModels with @Published properties
   - `Factory/` - Dependency injection
   - `ViewController` - UI components

4. **Navigation**
   - `Coordinator` - Scene navigation management

## Installation

Run the setup script to install templates:

```bash
./download_templates.sh
```

## Usage

1. Create a new scene in Xcode
2. Select "Scene" template
3. Enter the scene name (e.g., "Login")
4. The template will generate files following Clean Architecture principles
